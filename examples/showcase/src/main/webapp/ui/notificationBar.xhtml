Notification Bar

NotificationBar displays a multipurpose fixed positioned panel for notification either on top or bottom of the page. Any group of JSF components can be placed inside notification bar.

Source

<b:notificationBar position="top" effect="slide" widgetVar="bar" styleClass="top">
    <h:outputText value="What BlazeFaces has so far is just the beginning!" style="color:#FFCC00;font-size:36px;" />
</b:notificationBar>

<b:commandButton value="Show" onclick="bar.show()" type="button" />
<b:commandButton value="Hide" onclick="bar.hide()" type="button" />
