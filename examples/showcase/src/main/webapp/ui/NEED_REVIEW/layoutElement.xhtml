Layout - Element

Layouts can also be created at element level. Example below is a simple split panel implementation.

Left Center

Source

<b:layout style="min-width:400px;min-height:200px;" id="layout">
    <b:layoutUnit position="west" resizable="true" size="100" minSize="40" maxSize="200">
        Left
    </b:layoutUnit>

    <b:layoutUnit position="center">
        Center
    </b:layoutUnit>
</b:layout>
