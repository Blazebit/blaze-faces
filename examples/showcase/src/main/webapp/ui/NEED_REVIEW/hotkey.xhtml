HotKey

Hotkey is used to bind a key or a key combination to an client side or server side event via ajax. In first example use "left" and "right" keys to switch images and in second example you can do an ajax request with the (ctrl+shift+s) combination.

<h3>ImageSwitch Integration</h3>
<b:imageSwitch effect="fade" widgetVar="switcher" slideshowAuto="false">
    <b:graphicImage value="/images/nature1.jpg" />
    <b:graphicImage value="/images/nature2.jpg" />
    <b:graphicImage value="/images/nature3.jpg" />
    <b:graphicImage value="/images/nature4.jpg" />
</b:imageSwitch>

<b:hotkey bind="left" handler="switcher.previous();" />
<b:hotkey bind="right" handler="switcher.next();" />

<h3>Ajax Hotkey</h3>
<h:form>

    <b:hotkey bind="ctrl+shift+s" update="display" actionListener="\#{hotkeyController.action}"/>

    <h:panelGrid columns="2" style="margin-bottom:10px">
        <h:outputLabel for="firstname" value="Firstname:" />
        <b:inputText id="firstname" value="\#{partialProcessRequestBean.firstname}" />
    </h:panelGrid>

    <b:outputPanel id="display">
        <h:outputText value="Hello: \#{partialProcessRequestBean.firstname}" rendered="\#{not empty partialProcessRequestBean.firstname}"/>
    </b:outputPanel>

</h:form>
