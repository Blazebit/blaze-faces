ContextMenu - ImageSwitch

This example demonstrates how to attach a context-menu for a particular component. Right-click on imageSwitcher to see the available options.

ImageSwitcher with ContextMenu

Source

<b:imageSwitch id="images" effect="fade" widgetVar="gallery" slideshowAuto="false"
    style="width:400px">
    <b:graphicImage value="/images/nature1.jpg" />
    <b:graphicImage value="/images/nature2.jpg" />
    <b:graphicImage value="/images/nature3.jpg" />
    <b:graphicImage value="/images/nature4.jpg" />
</b:imageSwitch>

<b:contextMenu for="images" style="z-index:50">
    <b:menuitem value="Previous" url="javascript:void(0)" onclick="gallery.previous()" icon="ui-icon-triangle-1-w" />
    <b:menuitem value="Next" url="javascript:void(0)" onclick="gallery.next()" icon="ui-icon-triangle-1-e" />
    <b:menuitem value="Remove All" url="javascript:void(0)" onclick="$('\#images').fadeOut()" icon="ui-icon-close"/>
</b:contextMenu>
